# Sprint 409 - Example Feedback Comparison Note

## Purpose

Compare the DB pension feedback direction against other bundled examples.

## Comparison

- DB pension couple emphasizes pension income, survivor impact, and estate intent.
- Early-retired couple emphasizes a long pre-benefit runway, non-registered assets, and tax timing.
- Already-retired couple emphasizes withdrawal order, taxes, and survivor impact.

## Shared finding

All three examples should keep the first review focused on the same consumer questions: how much monthly spending is being reviewed, whether the plan remains funded, what money is left, and which tax/OAS diagnostics explain the trade-off.

## Boundary

This note supports feedback planning only. It does not start annual sequencing architecture.
