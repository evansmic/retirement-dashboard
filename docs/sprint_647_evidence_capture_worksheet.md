# Sprint 647: Evidence Capture Worksheet

## Purpose
Define a manual worksheet for recording outside-app review evidence.

## Fields
- Reviewer label: anonymous short label only.
- Plan type: built-in example, single-person local plan, or two-person local plan.
- First answer clarity: clear, partly clear, or unclear.
- Spending language clarity: clear, partly clear, or unclear.
- Broad withdrawal-family clarity: clear, partly clear, or unclear.
- Trust and privacy reaction: reassuring, neutral, or concerning.
- Main confusion note: one short plain-language note.
- Follow-up category: copy cleanup, input clarity, Details organization, deferred scope, or no change.

## Outcome
- Feedback notes stay structured enough to compare.
- The worksheet avoids collecting personal identifiers or plan files.

## Boundary
This sprint does not add a database, app telemetry, saved feedback, reviewer accounts, or in-product forms.
