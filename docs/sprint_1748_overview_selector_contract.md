S1748 Overview Selector Contract

Purpose
- Define selectors suitable for Overview.

Overview can consume
- Retirement answer summary.
- Spending capacity summary.
- Minimum-expense coverage summary.
- Runtime recommendation selection.
- Runtime funding trace.
- Trace review summary.
- Plan health and review summaries.

Boundary
- Overview should not consume planning, audit, readiness, dry-run, or package closeout selectors.
