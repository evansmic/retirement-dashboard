# V1 Checkpoint Google Form

Use this as the Google Form structure. Recommended form title:

**Canadian Retirement Planner V1 Checkpoint Feedback**

Recommended description:

> Please use example or fictional data only. This review is about product clarity, trust, and usability. It is not financial advice.

## Section 1: Tester Info

1. **Tester name or initials**
   - Short answer
   - Optional

2. **Tester type**
   - Multiple choice
   - Options:
     - Non-expert consumer
     - Financial/planning professional
     - Developer/model reviewer
     - Other

3. **Device used**
   - Multiple choice
   - Options:
     - Desktop/laptop
     - Tablet
     - Phone
     - Multiple devices

4. **Examples tested**
   - Paragraph
   - Prompt: “List the built-in examples or plan types you tested.”

## Section 2: First Results Screen

5. **Could you answer “Can I retire?” within 60 to 90 seconds?**
   - Linear scale 1-5
   - 1 = No, not at all
   - 5 = Yes, clearly

6. **What did the first Results screen tell you in your own words?**
   - Paragraph

7. **How clear was the spending estimate?**
   - Linear scale 1-5
   - 1 = Confusing
   - 5 = Clear and appropriately cautious

8. **Did spending language feel like a planning estimate rather than a guarantee?**
   - Multiple choice
   - Options:
     - Yes
     - Mostly
     - No
     - Not sure

9. **Were the top review actions useful?**
   - Linear scale 1-5
   - 1 = Not useful
   - 5 = Very useful

10. **What, if anything, made the first screen feel too dense or too light?**
    - Paragraph
