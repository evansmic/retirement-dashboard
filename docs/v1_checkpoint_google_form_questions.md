# V1 Checkpoint Google Form

Use this as the Google Form structure. Recommended form title:

**Canadian Retirement Planner V1 Checkpoint Feedback**

Recommended description:

> Please use example or fictional data only. This review is about product clarity, trust, and usability. It is not financial advice.
