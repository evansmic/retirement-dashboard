S1642 Recommendation Runtime Execution Package Closeout

Purpose
- Add a package closeout for recommendation runtime execution planning.
- Keep it as a readiness marker, not a selected recommendation.

Implemented boundary
- Package is complete only when the closeout is ready, block review is clear, and examples are ready.
- Package is blocked if any readiness input is blocked.
- The next broad step remains future recommendation runtime execution.

Deferred
- Selecting a recommendation candidate.
- Persisting optimizer or recommendation output.
- Showing recommendation output in UI.
