## Sprint 395

Sprint 395 drafts a short optimizer user-testing script.

Script:
- Start in Overview. Ask: What is the retirement answer?
- Ask: What spending number would you remember?
- Open Details. Ask: What does the first plan option mean?
- Ask: Is this advice, a command, or something to review?
- Ask: Which evidence row matters most: funded years, tax, OAS recovery, or money left?
- Ask: Does cash-wedge or flexibility language help, distract, or sound like an instruction?
- End by asking what should happen before annual account-level sequencing is considered.

Boundary:
- This script is for feedback collection only.
- It does not create saved feedback fields.
