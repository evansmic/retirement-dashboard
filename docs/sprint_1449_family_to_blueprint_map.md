# S1449 Family To Blueprint Map

The initial map is deliberately small:
- current plan -> baseline,
- spending repair -> small and larger floor repair,
- work timing -> one-year and two-year later-work comparisons,
- CPP/OAS timing -> review-only timing review,
- tax and estate -> review-only caveat review,
- broad withdrawal family -> review-only high-level review,
- home equity -> review-only or deferred reliance review,
- annual sequencing -> deferred marker only.

Only visible floor gaps open practical spending and work-timing blueprints.
