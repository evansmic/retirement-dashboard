# Sprint 343: Confusion Signal Guard

Sprint 343 adds confusion signals for broad withdrawal-family feedback.

## Change

- Flag when a user reads a broad family as a year-by-year withdrawal instruction.
- Flag when a user expects exact account amounts before reviewing the high-level trade-off.
- Flag when a user treats lower tax or higher money left as advice instead of plan-review evidence.

## Decision Use

Any of these signals should keep annual account-level sequencing deferred until copy or evidence is improved.
