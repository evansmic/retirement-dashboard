# Sprint 658: Review Scheduling Packet

## Purpose
Prepare a short packet for scheduling outside-app review sessions.

## Packet
- Ask for 30 to 45 minutes.
- Explain that the review is about clarity, trust, and privacy, not financial advice.
- Offer built-in examples if the reviewer does not want to use a local plan.
- Explain that no plan file needs to be sent.
- Confirm that notes will be anonymized and summarized manually.

## Outcome
- Review invitations can set expectations without overpromising product scope.
- Reviewers can participate without sharing private financial files.

## Boundary
This sprint does not create an intake form, mailing list, CRM, or account workflow.
